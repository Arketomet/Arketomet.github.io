Changelog and TODO

Changelog

12/17/2018:

  • The entirety of Kohi.la is now running via a Node.js web application! This will allow for implementation of an array of new features, including a potential rewrite of VastParser for added scalability and efficiency. Woo!

11/2/2018:

  • Added auto-response (\AUTO) to selection of backslash-escaped system messages.

11/11/2018:

  • Backslash-escaped system messages have been successfully implemented!
    • The "Began Trolling" and "Ceased Trolling" interface options have been removed and implemented into the new backslash-escaped system messages feature.
    • A ton of new system message options have been added with this feature.
  • Added a Help section to the interface to assist people with using Vast Parser.

11/9/2018:

  • Process of migrating all character attributes to a singular JSON file has been completed!
  • Quirk() now applies a quirk to a line based on what is saved for a character's JSON object instead of a switch/case statement. That sucked.
    • Consequentally, Quirk() now operates on a set list of quirk types derived from the least common denominator of all quirks. This includes: upperCase, lowerCase, affix, and replace to name a few.
  • Ancestors have been added! That's right: All named and/or titled Ancestors (To the point of public knowledge) have been added to the list of dialogue-able characters. Try em out!
  • Added more meta tags for that good, good Discord rich object presence.

10/31/2018:

  • Big changes are coming! I have begun internal changes to allow for custom character submissions.
  • Moved all unique character information into characters.json.
  • Adapted Character() programming logic to reflect this change.

10/29/2018:

  • Condensed character quirks into a single Quirk() function. This is futureproofing for adding new character quirks. It also condensed the VastParser.js line count by over 100 lines and didn't break the program. Holla holla get $$$.
  • I'm a dumb and broke prefixes while toggled into Edolon's log theme. I fixed it.
  • Made the Skorpelog and Dialoguelog log types more robust.
    • In Skorpelog mode, all prefixes are automatically converted to the most appropriate Skorpe handle acronym.
    • In Dialoguelog mode, all prefixes are automatically converted to the most appropriate character name.
  • Separated AO3 Mode from the list of log types, following the Skorpelog and Dialoguelog changes, so new functionality can be used while writing logs for AO3.

10/28/2018:

  • Added colors for all known characters.

10/27/2018:

  • Added a Changelog!
  • Backend changes to Trolling() function to allow for future backslash-escaped command implementation.
  • Added an AO3 Mode just for Kate. <3
  • Added Laivan's quirk.
